📚 Library Management System

A full-stack web application designed to simplify library operations by managing books, issued/returned records, authentication, and fines.

👩‍💻 Developers

Thahaseen Gulam

🚀 Features
✅ User & Admin Authentication

Secure login and registration
Email verification
Role-based access (Admin / User)

📘 Book Management

Add, update, delete, and view books
Maintain book availability status
Search and filter books

🔄 Issue/Return Management

Issue books to users
Track return dates
Auto fine calculation for late returns

📊 Admin Dashboard

Overview of total books, issued books, returned books
Manage users and records from one place

🎨 Front-End

Clean and responsive UI
Built using HTML, CSS, JavaScript

🗄️ Back-End

Handles all CRUD operations
Validations for issuing, returning, and user roles

🏗️ Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Java / PHP / Node.js (update based on your project)
Database: MySQL / MongoDB (update based on your project)
